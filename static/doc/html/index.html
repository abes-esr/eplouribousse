Documentation d’eplouribousse¶

eplouribousse [eˈpluːrɪbʊs] est une application web pour faciliter le dédoublonnement de revues entre un ensemble de bibliothèques.

Jetez un coup d’oeil au site strasbourgeois pour vous faire une idée.
